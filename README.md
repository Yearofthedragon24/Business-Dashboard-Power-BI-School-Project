# 📊 Business Dashboard (Power BI - School Project)

An early Power BI project demonstrating data analysis across sales, HR, and finance.

## 📊 Project Overview

This Power BI dashboard was developed as an early school project to analyze business performance across multiple areas.

It provides insights into sales, HR, and financial data, helping to understand overall business operations and key metrics.

## 🚀 Key Features


- Sales performance analysis over time  
- HR insights (employee distribution, salaries, working hours)  
- Financial reporting (revenue, costs, profit)  
- Interactive filters and slicers  
- Multi-page dashboard (Sales, HR, Financial overview)


## 📈 Business Insights

The dashboard highlights important patterns in business data:

- Sales trends show variations across months and periods  
- HR data provides visibility into workforce structure and workload  
- Financial metrics help track profitability and cost distribution  

These insights support a better understanding of business performance.
