# 📊 Business Dashboard (Power BI - School Project)

An early Power BI project analyzing sales, HR, and financial performance.

## 📊 Project Overview

This Power BI dashboard analyzes business performance across sales, HR, and financial data.

It was developed as an early project to demonstrate data analysis and visualization skills.

## 🚀 Key Features
- Sales performance analysis over time  
- HR insights (employee distribution, salaries, working hours)  
- Financial reporting (revenue, costs, profit)  
- Interactive filters and slicers  
- Multi-page dashboard (Sales, HR, Financial overview)


## 📈 Business Insights

The dashboard highlights important patterns in business data:

- Sales trends show variations across months and periods  
- HR data provides visibility into workforce structure and workload  
- Financial metrics help track profitability and cost distribution  

These insights support a better understanding of business performance.

## 📚 What I Learned

- How to build multi-page dashboards in Power BI  
- Data modeling and creating relationships between tables  
- Writing DAX measures for KPIs and calculations  
- Designing interactive and user-friendly reports  

## 🛠 Tools Used

- Power BI  
- DAX  
- Data modeling
